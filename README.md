# Myntra Sales Analysis Dashboard

## Project Overview
This project focuses on analyzing Myntra sales data using Microsoft Excel to uncover customer behavior patterns and business insights. An interactive dashboard was created to visualize key performance indicators and support data-driven decision-making.

## Objectives
- Analyze Myntra sales performance
- Identify top-performing states
- Understand customer purchasing behavior by gender
- Examine shopping patterns across different age groups
- Compare orders and sales trends

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning Techniques
- Dashboard Design

## Key Tasks Performed
- Filtered Myntra-specific data from a multi-store dataset
- Cleaned and formatted raw sales data
- Sorted and organized data for analysis
- Identified the Top 10 states based on sales performance
- Analyzed purchases by men vs. women customers
- Performed age-wise and gender-wise customer analysis
- Compared order volume with sales performance
- Created an interactive dashboard using charts and slicers

## Key Insights
- Identified the highest revenue-generating states
- Determined the dominant customer segment by age and gender
- Highlighted shopping trends and customer preferences
- Provided actionable insights through visual reports
